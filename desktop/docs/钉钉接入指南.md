# 钉钉接入指南（自动化中心）

## 已上线：群通知（A 期）

mcn-ai 会在以下时刻自动推送消息到你的钉钉群：
- **投递箱处理完成/失败**（含本轮处理的文件名列表）
- **AI 产物生成**（Word/Excel/PDF/PPT 落盘时）

### 配置步骤（2 分钟，群主/管理员操作）

1. 打开目标钉钉群 → 右上角「群设置」→「智能群助手」→「添加机器人」→ 选「**自定义**」
2. 起名（如 mcn-ai）；安全设置勾选「**加签**」，复制出现的 `SEC…` 密钥
3. 点完成，复制生成的 Webhook 地址（`https://oapi.dingtalk.com/robot/send?access_token=…`）
4. 回到 mcn-ai → 设置 → 「自动化 · 钉钉通知」：粘贴 Webhook 与加签密钥 → 点「发送测试消息」
5. 钉钉群里收到「钉钉接入成功 🎉」即完成；两个开关可分别控制通知类型

> 安全设置若选了「自定义关键词」而不是加签：关键词必须包含 `mcn-ai`，且密钥留空即可。

## 规划中：钉钉里问知识库（B 期）

目标：员工在钉钉群 @机器人 提问，AI 检索公司知识库直接回答（带引用）。

技术路径已验证可行：钉钉开放平台企业内部应用（HTTP 回调模式）→ 我们的
API 服务器（www.makeupai.top）→ 三层知识检索 + 模型回答 → 经 sessionWebhook 回帖。

需要管理员先完成（一次性，约 15 分钟）：
1. dingtalk.com 开放平台 → 创建「企业内部应用」→ 添加「机器人」能力
2. 消息接收模式选 **HTTP**，回调地址填 `https://www.makeupai.top/api/dingtalk/callback`
3. 把应用的 AppKey / AppSecret 交给开发（配到 Vercel 环境变量）

完成后通知开发联调（服务端代码半天内可就绪）。
